# 🎯 Word Guess Game

A simple Python word guessing game made using basic Python concepts.

## 📌 Features

- Random words loaded from `words.txt`
- Colored terminal output
- Shows word length before starting
- Letter checking system:
  - `[D]` → correct letter & correct position
  - `(D)` → correct letter but wrong position
  - ` D ` → letter not found
- Exception handling
- Beginner friendly code
